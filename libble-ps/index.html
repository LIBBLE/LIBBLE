LIBBLE-PS

Introduction

LIBBLE-PS is part of the Project LIBBLE. The current version of LIBBLE-PS includes the following machine learning algorithms:

  • Classification
    • Logistic Regression (LR)

LIBBLE-PS uses SCOPE algorithm on Parameter Server framework. It uses multi Servers and Workers to accelerate. We choose logistic regression (LR) with a L2-norm regularization term to evaluate LIBBLE-PS and other methods. The result on webspam dataset is shown below. All the methods use 16 Workers.
